git reset HEAD --hard -> permet supprimer les modifications et revenir à l'état de l'ancien commit 

git reset HEAD~1 -> permet d'annuler le commit sans modifier les fichiers avec nos modifications

git reset HEAD~1  --hard -> la combinaison des deux explications 


Git remote and Origin ( lien du projet ) -> Ajout du projet local vers le projet distant 

Git remote -v -> Apercu du lien vers le projet distant 

Git branch -M main. -> renommer la branche principal en main ( nom de variable )
